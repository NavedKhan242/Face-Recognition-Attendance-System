
# 🎯 Face Recognition Attendance System

A Python-based Face Recognition Attendance System that automates attendance marking using real-time face detection and recognition. The application features an intuitive Tkinter GUI for student management, dataset generation, model training, and attendance tracking.

---

## 📌 Features

- 👤 Student Registration & Management
- 📷 Face Dataset Generation
- 🧠 Face Recognition using OpenCV
- 🎓 Automatic Attendance Marking
- 📊 Attendance Record Management
- 🖥️ User-Friendly Tkinter GUI
- ⚡ Real-Time Face Detection
- 🔒 Login System

---

## 🛠️ Tech Stack

- Python
- OpenCV
- Tkinter
- NumPy
- Pandas
- Pillow (PIL)
- SQLite
- Haar Cascade Classifier

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/NavedKhan242/Face-Recognition-Attendance-System.git
```

### 2. Go to the project directory

```bash
cd Face-Recognition-Attendance-System
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the project

```bash
python main.py
```

---

## 📋 Requirements

- Python 3.x
- OpenCV

- Pandas
